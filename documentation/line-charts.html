Line Charts

Single series, multi-series, filled area, trendlines, and Bézier curves.

Basic Line Chart

The default jqPlot series renderer draws a line chart. Pass a single inner array for one series, or multiple inner arrays for multiple series. Data points can be plain numbers (x inferred from index) or explicit [x, y] pairs. 

$.jqplot('chart', [
  [14, 22, 18, 31, 27, 35]   // one series — x = 1,2,3,4,5,6
], {
  series: [{
    label:     'Revenue',
    color:     '#2980b9',
    lineWidth:  2.5,
    markerOptions: { size: 7, style: 'filledCircle' }
  }],
  legend: { show: true, location: 'nw' },
  axes: {
    xaxis: { label: 'Month' },
    yaxis: {
      label:         'Revenue ($k)',
      labelRenderer: $.jqplot.CanvasAxisLabelRenderer,
      tickOptions:   { formatString: '$%dk' }
    }
  }
});

Multi-Series Line Chart

Pass multiple inner arrays — one per series. The series array provides per-series overrides; index 0 in series corresponds to index 0 in the data array. 

var revenue = [14, 22, 18, 31, 27, 35];
var cost    = [10, 15, 13, 19, 17, 22];
var profit  = revenue.map(function(r,i){ return r - cost[i]; });

$.jqplot('chart', [revenue, cost, profit], {
  series: [
    { label: 'Revenue', color: '#2980b9' },
    { label: 'Cost',    color: '#e74c3c' },
    { label: 'Profit',  color: '#27ae60' }
  ],
  legend: { show: true, location: 'nw' }
});
The series array is positional — element 0 styles the first data array, element 1 styles the second, and so on.

Filled Area Chart

Set fill: true on a series to shade the area below the line. fillAlpha controls opacity (0–1). Works well for showing trends where the area under the curve has meaning. 

$.jqplot('chart', [revenueData], {
  series: [{
    label:     'Annual Revenue',
    color:     '#8e44ad',
    lineWidth:  2.5,
    fill:       true,
    fillAlpha:  0.15,          // subtle fill
    markerOptions: { size: 8, style: 'filledCircle' }
  }]
});

Trendline

The trendline plugin adds a best-fit regression line to any series. Include jqplot.trendline.min.js and add a trendline object to the series configuration. 

<script src="dist/plugins/jqplot.trendline.min.js"></script>

$.jqplot('chart', [salesData], {
  series: [{
    label: 'Monthly Sales',
    color: '#2980b9',
    trendline: {
      show:      true,
      color:     '#e74c3c',
      lineWidth:  2,
      label:     'Trend'
    }
  }]
});

Bézier Curve Chart

BezierCurveRenderer draws smooth curves between data points using cubic Bézier interpolation. Useful when the underlying function is smooth and linear interpolation looks jagged. 

<script src="dist/plugins/jqplot.BezierCurveRenderer.min.js"></script>

$.jqplot('chart', [seriesA, seriesB], {
  seriesDefaults: {
    renderer: $.jqplot.BezierCurveRenderer,
    showMarker: false
  },
  series: [
    { label: 'Product A', color: '#2980b9' },
    { label: 'Product B', color: '#e67e22' }
  ]
});

Y-Axis Auto-Scaling

Avoid hardcoding min and max unless you have a specific reason. jqPlot autoscales from the data range by default. Use padMin and padMax to add breathing room without clipping: 

yaxis: {
  // No min/max — autoscale from data
  padMin: 1.03,   // 3% below data minimum
  padMax: 1.03,   // 3% above data maximum
  tickOptions: { formatString: '%d' }
}
Setting min: 0 on a y-axis whose data can go negative will clip the negative values silently. Always check the actual data range before hardcoding axis bounds.
