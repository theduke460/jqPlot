Bar Charts

Vertical bars, horizontal bars, grouped multi-series bars, and value labels.

Required Plugin

All bar charts require BarRenderer and usually CategoryAxisRenderer:

<script src="dist/plugins/jqplot.barRenderer.min.js"></script>
<script src="dist/plugins/jqplot.categoryAxisRenderer.min.js"></script>

Vertical Bar Chart — Single Series

Set seriesDefaults.renderer to $.jqplot.BarRenderer and the x-axis renderer to $.jqplot.CategoryAxisRenderer. The ticks array provides category labels. 

var sales  = [48, 63, 71, 55, 82, 76];
var months = ['Jan','Feb','Mar','Apr','May','Jun'];

$.jqplot('chart', [sales], {
  seriesDefaults: {
    renderer:        $.jqplot.BarRenderer,
    rendererOptions: { fillToZero: true, barPadding: 4, barMargin: 10 }
  },
  series: [{ label: 'Sales ($k)', color: '#2980b9' }],
  legend: { show: true, location: 'nw' },
  axes: {
    xaxis: {
      renderer: $.jqplot.CategoryAxisRenderer,
      ticks:    months
    },
    yaxis: {
      label:         'Sales ($k)',
      labelRenderer: $.jqplot.CanvasAxisLabelRenderer,
      min:  0,
      tickOptions: { formatString: '$%dk' }
    }
  }
});

Grouped Bar Chart — Multiple Series

Pass multiple series arrays. jqPlot automatically applies a _barNudge offset to place bars side-by-side within each category. Use barPadding to control the gap between bars in a group, and barMargin for the gap between groups. 

var north  = [48, 63, 71, 55, 82, 76];
var south  = [39, 51, 68, 44, 73, 61];
var months = ['January','February','March','April','May','June'];

$.jqplot('chart', [north, south], {
  seriesDefaults: {
    renderer:        $.jqplot.BarRenderer,
    rendererOptions: { barPadding: 5, barMargin: 16 }
  },
  series: [
    { label: 'North Region', color: '#2e86c1' },
    { label: 'South Region', color: '#e59866' }
  ],
  legend: { show: true, location: 'nw' },
  axes: {
    xaxis: {
      renderer: $.jqplot.CategoryAxisRenderer,
      ticks:    months,
      tickOptions: { angle: -20 }
    },
    yaxis: { min: 0, tickOptions: { formatString: '$%dk' } }
  }
});
_barNudge pitfall: Use one multi-point series (not N single-point series) when you want one bar per category. Multiple single-point series causes jqPlot to treat them as a grouped chart and apply nudge offsets that push bars off-canvas. See Technical Notes Issue 5.

Horizontal Bar Chart

Add barDirection: 'horizontal' to rendererOptions. The y-axis becomes the category axis; the x-axis carries the numeric scale. Data format is [xValue, yIndex] pairs, where yIndex maps to a tick position. 

// Single series — use varyBarColor for per-bar colors without grouped nudge
$.jqplot('chart', [[
  [88, 2], [74, 3], [62, 4], [91, 5], [47, 6]
]], {
  seriesDefaults: {
    renderer:        $.jqplot.BarRenderer,
    rendererOptions: {
      barDirection: 'horizontal',
      varyBarColor:  true,
      barWidth:      22
    }
  },
  seriesColors: ['#27ae60','#2980b9','#8e44ad','#e67e22','#c0392b'],
  axes: {
    xaxis: {
      label:  'Score',
      min: 0, max: 100,
      tickOptions: { formatString: '%d' }
    },
    yaxis: {
      // Blank guard ticks at positions 1 and 7 prevent bars touching edges.
      // Named ticks at 2–6; bar data uses yIndex 2–6 to match.
      renderer: $.jqplot.CategoryAxisRenderer,
      ticks: ['', 'React', 'Angular', 'Node.js', 'Python', 'Rust', '']
    }
  }
});
Guard ticks: Add empty strings at the start and end of the ticks array. This shifts named categories to positions 2…N-1. Bar data y-indices must match these positions exactly.

Value Labels Inside Bars

Use the pointLabels plugin with escapeHTML: false to inject styled HTML labels inside bars. For horizontal bars use location: 'w' (anchors the label's right edge at the bar tip). 

<script src="dist/plugins/jqplot.pointLabels.min.js"></script>

// Build styled HTML label strings
var labelStyle = 'font-weight:bold;color:#fff;' +
  'text-shadow:1px 1px 0 #000,-1px -1px 0 #000,' +
              '1px -1px 0 #000,-1px 1px 0 #000;';

function hbarLabel(val) {
  return '<span style="' + labelStyle + '">' + val + '%</span>';
}

// Per-series pointLabels with explicit labels array:
series: [{
  pointLabels: {
    show:          true,
    location:      'w',        // right-edge of label at bar tip
    xpadding:      6,
    escapeHTML:    false,      // allow HTML in label strings
    edgeTolerance: -100,       // prevent suppression on short bars
    labels: [hbarLabel(88), hbarLabel(74), hbarLabel(62)]
  }
}]
