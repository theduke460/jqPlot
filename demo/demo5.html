Log Axis · Category Axis — Vertical Bar · Category Axis — Grouped Horizontal Bar · Point Labels & Highlighter

Line Chart — Log Axis (Bacteria Population Growth)

LogAxisRenderer maps the y-axis to a base-10 logarithmic scale. Each major gridline represents a 10× increase. Useful whenever data spans several orders of magnitude.

Vertical Bar Chart — Category Axis (Q1 Department Budget)

CategoryAxisRenderer assigns one tick per data point. Labels come from the ticks array; no numeric scale is implied.

Horizontal Bar Chart — Category Axis (Developer Satisfaction by Tech Stack)

Two series (2023 vs 2024) produce grouped horizontal bars — one pair per tech stack. Each series gets its own legend color. barDirection: 'horizontal' pivots the chart; CategoryAxisRenderer on the y-axis provides the category labels.

Line Chart — Point Labels & Highlighter (Monthly Active Users)

PointLabels plugin annotates each data point with its value. Highlighter plugin shows a tooltip on hover — move your mouse over any marker.
