jqPlot Demo 6

Highlighter · Cursor · Canvas Overlay · Draggable · Category Axis · Date Axis · Log Axis

Server Response Times (ms)

Highlighter

Hover over any data point to see a formatted tooltip showing the exact server and response time. The highlighter plugin enlarges the nearest marker and renders the tooltip via tooltipAxes and formatString.

CPU Utilization — Three Cores (%)

Cursor

Move your mouse across the chart to track the crosshair cursor. The cursor plugin renders a live x/y readout inside the plot area. showTooltip:true + followMouse:true keeps the label beside the crosshair intersection.

Stock Price with Target Zones

Canvas Overlay

The canvasOverlay plugin draws arbitrary lines on top of the chart after it renders. Here a solid green buy zone line, a solid red sell zone line, and a dashed orange analyst target line are added without touching the series data.

Sensor Readings — Drag to Edit

Draggable
Drag any point to edit its value…

The draggable plugin makes each marker interactively moveable. The output strip above updates in real time via the jqplotDataChange event as points are dragged.

Monthly Sales by Region — Q1

Category Axis

CategoryAxisRenderer assigns one evenly-spaced slot per label on the x-axis — no numeric scale is implied. Two series (North, South) produce grouped bars with automatic _barNudge offset. Labels come from the ticks array.

Annual Revenue 2015–2024

Date Axis

DateAxisRenderer parses date strings on the x-axis and formats tick labels via strftime patterns. Here '%Y' shows only the year. Data is passed as ['YYYY-MM-DD', value] pairs; jqPlot converts them to numeric timestamps internally.

RF Signal Strength vs Distance

Log Axis

Both axes use LogAxisRenderer — a log-log plot. RF signal power follows an inverse-square law with distance, producing a straight line on a log-log scale. A second series shows measured values with real-world multipath fading.

Product Revenue Timeline — Two-Line Ticks · Time Ranges · Time Events

Date Axis Enhanced

Demonstrates three dateEnhancement plugin features: (1) two-line tick labels using \n in formatString; (2) shaded dateRanges with color, opacity, and label; (3) dateEvents shown as labeled markers with randomized Y positions to reduce label collision — each event has a deterministic seed for stable placement.

Historical Timeline 1910–2025 — Wars · Recessions · NASA Programs · Key Events

Date Axis Enhanced

Green ranges: major US wars. Blue ranges: US economic recessions (post-1960). Yellow ranges: NASA human spaceflight programs (Gemini, Apollo, Space Shuttle). Ranges narrower than 100px are automatically collapsed into diamond markers (collapsed range) rather than cluttered inline labels. Circle markers indicate point-in-time events. Five key historical events are shown as circles.
