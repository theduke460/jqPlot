Specialized Charts

Bubble, candlestick, meter gauge, funnel, pyramid, block, and Mekko charts.

Bubble Chart

Data points are [x, y, radius, {label}] four-element arrays. The third element is the bubble radius in data units; the optional fourth element is an object that may include a label string shown inside the bubble. 

<script src="dist/plugins/jqplot.bubbleRenderer.min.js"></script>

var bubbles = [
  [25, 30, 8,  { label: 'Alpha' }],
  [40, 55, 12, { label: 'Beta'  }],
  [60, 20, 6,  { label: 'Gamma' }]
];

$.jqplot('chart', [bubbles], {
  seriesDefaults: {
    renderer: $.jqplot.BubbleRenderer,
    rendererOptions: {
      bubbleAlpha:      0.7,
      showLabels:       true,
      bubbleGradients:  false
    }
  }
});

Candlestick (OHLC) Chart

Data points are [date, open, high, low, close] five-element arrays. Set candleStick: true for filled candles; false for classic OHLC tick marks. Requires DateAxisRenderer on the x-axis. 

<script src="dist/plugins/jqplot.ohlcRenderer.min.js"></script>
<script src="dist/plugins/jqplot.dateAxisRenderer.min.js"></script>

var ohlc = [
  ['2024-01-02', 182.5, 188.2, 181.0, 186.4],
  ['2024-01-03', 186.4, 189.5, 184.3, 185.9],
  ['2024-01-04', 185.9, 187.8, 183.1, 184.7]
];

$.jqplot('chart', [ohlc], {
  series: [{
    renderer:        $.jqplot.OHLCRenderer,
    rendererOptions: { candleStick: true }
  }],
  axes: {
    xaxis: {
      renderer:    $.jqplot.DateAxisRenderer,
      tickOptions: { formatString: '%b %e' }
    }
  }
});

Meter Gauge

A single data point drives the gauge needle. intervals defines colored arc segments; intervalColors assigns colors to each. The label shown inside the gauge is the raw value. 

<script src="dist/plugins/jqplot.meterGaugeRenderer.min.js"></script>

$.jqplot('gauge', [[72]], {
  seriesDefaults: {
    renderer: $.jqplot.MeterGaugeRenderer,
    rendererOptions: {
      min:            0,
      max:            100,
      intervals:      [33, 66, 100],
      intervalColors: ['#27ae60','#f39c12','#e74c3c'],
      label:          'CPU %',
      labelPosition:  'below'
    }
  }
});
Nginx caching bug: With expires max headers, browsers may return the parent container height for outerHeight() calls on freshly-inserted position:absolute elements, producing a negative radius and an IndexSizeError. This fork applies a defensive height clamp in the renderer. See Technical Notes Issue 2.

Funnel Chart

Data is an array of [label, value] pairs. The renderer sorts them descending automatically. Useful for sales pipelines and conversion funnels. 

<script src="dist/plugins/jqplot.funnelRenderer.min.js"></script>

var pipeline = [
  ['Leads',      1240],
  ['Qualified',   820],
  ['Proposals',   410],
  ['Negotiation', 195],
  ['Closed',       88]
];

$.jqplot('chart', [pipeline], {
  seriesDefaults: {
    renderer: $.jqplot.FunnelRenderer,
    rendererOptions: {
      sectionMargin: 6,
      widthRatio:    0.35,
      showDataLabels: true
    }
  }
});
See demo3 →
Funnel segment labels render over colored fills. Apply the white/black-outline label pattern via .jqplot-funnel-series.jqplot-data-label in your stylesheet.

Pyramid Chart

Used for population pyramids. Data format is [yIndex, value] — note the order: y-index (age group) first, bar width (population) second. Two series (male/female) are passed to create mirrored bars. 

<script src="dist/plugins/jqplot.pyramidAxisRenderer.min.js"></script>
<script src="dist/plugins/jqplot.pyramidGridRenderer.min.js"></script>
<script src="dist/plugins/jqplot.pyramidRenderer.min.js"></script>
<!-- Load dependency scripts BEFORE the renderer that depends on them -->

var female = [[1,4820],[2,4650],[3,4210],[4,3890],[5,3120]];
var male   = [[1,4640],[2,4520],[3,4080],[4,3750],[5,2990]];

$.jqplot('chart', [female, male], {
  seriesDefaults: {
    renderer: $.jqplot.PyramidRenderer
  },
  series: [
    { label: 'Female', color: '#e91e8c' },
    { label: 'Male',   color: '#1565c0' }
  ],
  axes: {
    yaxis: {
      ticks: ['0–9','10–19','20–29','30–39','40–49']
    }
  }
});
Plugin load order: pyramidAxisRenderer and pyramidGridRenderer must be loaded before pyramidRenderer. Loading in the wrong order triggers a silent Ajax fallback that fails with a 404.

Block Chart

Renders text blocks on a grid. Useful for categorical status dashboards. The background color must be set at the series level via rendererOptions.css.background — per-point CSS is applied after init() and is invisible to the legend color system. 

<script src="dist/plugins/jqplot.blockRenderer.min.js"></script>

var healthy  = [['web-01','OK'],['web-02','OK'],['db-01','OK']];
var warning  = [['cache-01','WARN']];
var critical = [['db-02','CRIT']];

$.jqplot('chart', [healthy, warning, critical], {
  seriesDefaults: {
    renderer: $.jqplot.BlockRenderer
  },
  series: [
    { label: 'Healthy',
      rendererOptions: { css: { background: '#27ae60', padding: '5px 7px' } } },
    { label: 'Warning',
      rendererOptions: { css: { background: '#e67e22', padding: '5px 7px' } } },
    { label: 'Critical',
      rendererOptions: { css: { background: '#c0392b', padding: '5px 7px' } } }
  ]
});
See demo4 →
Block cell labels render over colored fills. Apply the white/black-outline label pattern via .jqplot-blockCanvas div in your stylesheet.

Mekko (Marimekko) Chart

A stacked bar chart where bar widths are also proportional to a value. Each series is one category; data points are [x, value] pairs representing segments within that category's bar. 

<script src="dist/plugins/jqplot.mekkoRenderer.min.js"></script>
<script src="dist/plugins/jqplot.mekkoAxisRenderer.min.js"></script>

$.jqplot('chart', [northAmerica, europe, asiaPac], {
  seriesDefaults: {
    renderer:        $.jqplot.MekkoRenderer,
    rendererOptions: { borderColor: '#fff', showBorders: true }
  },
  axes: {
    xaxis: { renderer: $.jqplot.MekkoAxisRenderer },
    yaxis: { min: 0, max: 100 }
  }
});
